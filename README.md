# Mini Pokédex

## О проекте

**Mini Pokédex** - учебный покедекс (справочник покеомнов) с разделением на фронт (Стримлит) и бек(FastApi)

Пользователь выбирает покемона из списка, после чего приложение отображает его основную информацию: изображение, описание, типы, рост, вес, способности, базовые характеристики и паутинную диаграмму характеристик.

В качестве источника данных используется открытое API Pokémon — PokéAPI.

## Архитектура проекта

Проект состоит из двух независимых частей:

### Frontend (Streamlit)

Frontend отвечает за пользовательский интерфейс:

* отображение списка покемонов
* выбор покемона
* отправку запросов в backend
* отображение карточки покемона
* построение таблицы характеристик
* построение паутинной диаграммы

### Backend (FastAPI)

Backend выполняет роль промежуточного слоя между frontend и PokéAPI:

* принимает запросы от Streamlit;
* обращается к PokéAPI;
* извлекает только необходимые данные;
* преобразует исходный JSON;
* возвращает frontend очищенный и удобный для отображения ответ.

## Установка зависимостей

Создайте виртуальное окружение и установите зависимости:

```bash
pip install -r requirements.txt
```

---

## Запуск backend

Из корневой директории проекта выполните команду:

```bash
uvicorn backend.main:app --reload
```

После запуска backend будет доступен по адресу:

```text
http://127.0.0.1:8000
```

---

## Запуск frontend

В отдельном терминале выполните:

```bash
streamlit run frontend/app.py
```

После запуска приложение откроется в браузере автоматически.
